# Datasets

This folder contains sample datasets used for lightweight analysis and dashboard prototypes.

## Files

### 1. ticket_data.csv
Sample ticket-level data including:
- categories
- priorities
- timestamps
- agents
- CSAT

Useful for ticket volume analysis, SLA metrics, and agent performance.

### 2. funnel_data.csv
A synthetic SaaS funnel showing:
- users per stage
- conversions
- drop-offs

Useful for conversion analysis and identifying bottlenecks.

### 3. spend_data.csv
Corporate spend data including:
- department
- category
- amount
- city

Useful for spend segmentation and insights.
